# 🚀 Smart Student Financial Intelligence Dashboard

A full-stack web application designed to go beyond simple expense tracking by providing **data-driven financial insights**, **behavior analysis**, and **predictive recommendations** for students.

---

## 💡 Project Overview

This system transforms basic expense tracking into a **financial intelligence platform** that helps users:

- Understand spending behavior
- Monitor budget usage
- Detect unusual expenses
- Predict future financial trends
- Make smarter financial decisions

---

## ✨ Key Features

### 📊 Interactive Dashboard
- Power BI-style modern UI
- Real-time analytics and data visualization

### 📈 Spending Trend Analysis
- Daily expense tracking
- Cumulative spending line chart
- Visual comparison of funds vs expenses

### 🎯 Budget Management
- Monthly budget goal tracking
- Budget usage percentage
- Remaining balance calculation
- Progress bar visualization

### 🧠 Smart Insights Engine
- Automatically generates insights such as:
  - Highest spending category
  - Overspending alerts
  - Budget usage warnings
  - Remaining balance analysis

### ⚠️ Anomaly Detection
- Detects unusual transactions based on spending patterns

### ⏳ Financial Forecasting
- Predicts how long your funds will last
- Estimates future spending trends

### 🔐 Authentication System
- Secure login & registration using JWT

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Recharts (Data Visualization)
- CSS (Custom Power BI-style UI)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Atlas)

### Authentication
- JSON Web Tokens (JWT)
